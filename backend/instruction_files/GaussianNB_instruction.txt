MODEL_SELECTION_REASONING:
GaussianNB should be prioritized for low-dimensional, purely numerical classification datasets where features follow approximately Gaussian distributions and classes are reasonably balanced. It is particularly effective for biomedical classification tasks (e.g., breast cancer diagnosis) and well-structured tabular datasets (e.g., wine, iris) where feature distributions per class are compact and separable. It serves well as a fast, interpretable baseline with near-zero configuration overhead.

MODEL_AVOIDANCE_REASONING:
Avoid GaussianNB for datasets dominated by categorical features, as the Gaussian likelihood assumption is fundamentally violated. Also avoid for severely imbalanced classification problems where the uniform prior assumption introduces systematic bias. GaussianNB underperforms on datasets with non-Gaussian numerical distributions (e.g., heavy-tailed or multimodal), and should not be used as a final model when high accuracy is required on mixed categorical-numerical data without appropriate encoding strategies.
