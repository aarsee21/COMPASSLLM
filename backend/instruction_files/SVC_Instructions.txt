HYPERPARAMETER_NOTES:

MODEL_SELECTION_REASONING:
SVC should be prioritized for low-to-medium dimensional classification tasks with scaled numerical features and clear class margins. RBF kernel makes it effective for non-linear boundaries. Performs best on balanced datasets with 2–3 high-signal numerical features and encoding applied for any categorical inputs.

MODEL_AVOIDANCE_REASONING:
Avoid SVC for high-dimensional datasets, large-scale data (inference is slow without scaling), imbalanced class distributions without compensation, or when features are unscaled. Also underperforms on single weak or sparse features and purely categorical spaces without strong discriminative signals.
